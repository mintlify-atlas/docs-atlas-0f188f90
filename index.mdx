<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold dark:text-white text-gray-900 mb-4">
    Quick start
  </h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">
    Get up and running with Neutronium in minutes
  </p>

  <Steps>
    <Step title="Install dependencies">
      Install the required build tools on Ubuntu/Debian:

      ```bash
      sudo apt update
      sudo apt install build-essential cmake ninja-build
      ```

      Make sure your `cmake` version is at least 3.28.
    </Step>

    <Step title="Clone and configure">
      Clone the Neutronium repository and set up the build scripts:

      ```bash
      git clone https://github.com/Thibault-Monnier/neutronium.git
      cd neutronium
      chmod +x configure-cmake.sh compile.sh run.sh test.sh
      ./configure-cmake.sh
      ```
    </Step>

    <Step title="Write your first program">
      Create a simple Neutronium program:

      ```neutronium hello.nt
      extern fn print_num(num: int);

      fn main(): {
          let message = 42;
          print_num(message);
      }
      ```
    </Step>

    <Step title="Compile and run">
      Use the `run.sh` script to compile and execute your program:

      ```bash
      ./run.sh hello.nt
      ```

      The compiler will build your program and run it, outputting `42` to the console.
    </Step>
  </Steps>
</div>
