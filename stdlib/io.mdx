Neutronium's standard library provides I/O functions for printing output to stdout. These functions are implemented in assembly and available through external declarations.

## Declaring I/O functions

To use I/O functions, declare them with the `extern fn` keyword:

```neutronium
extern fn print_c(char: int);
extern fn print_num(num: int);
```

## print_c

Prints a single character to stdout.

```neutronium
extern fn print_c(char: int);
```

<ParamField path="char" type="int" required>
  The character to print, represented as an integer ASCII value
</ParamField>

### Example

```neutronium
extern fn print_c(char: int);

fn main(): {
    # Print 'H'
    print_c(72);
    
    # Print 'i'
    print_c(105);
    
    # Print newline
    print_c(10);
}
```

## print_num

Prints an integer number to stdout. Handles both positive and negative numbers.

```neutronium
extern fn print_num(num: int);
```

<ParamField path="num" type="int" required>
  The integer number to print. Can be positive, negative, or zero.
</ParamField>

### Example

```neutronium
extern fn print_num(num: int);
extern fn pow(a: int, b: int) -> int;
extern fn abs(val: int) -> int;

fn mixLogic(flag: bool, base: int): {
    let value = pow(base, 2);
    if flag: {
        print_num(value);
    } else: {
        print_num(abs(-value));
    }
}

fn main(): {
    # Calculate and print 2^8
    let p = pow(2, 8);
    print_num(p);  # Outputs: 256
    
    # Print computed results
    mixLogic(true, 9);   # Outputs: 81
    mixLogic(false, 3);  # Outputs: 9
}
```

## Real-world usage

Here's a complete example from the Neutronium benchmark tests:

```neutronium
extern fn print_num(num: int);
extern fn pow(a: int, b: int) -> int;
extern fn mod(a: int, b: int) -> int;
extern fn max(a: int, b: int) -> int;
extern fn abs(val: int) -> int;

fn computeValue(x: int, y: int) -> int: {
    let diff = abs(x - y);
    let power = pow(diff, 2);
    return power / 2;
}

fn main(): {
    # Test calls to stdlib functions
    let p = pow(2, 8);
    let m = mod(p, 7);
    let v = max(p, m);
    print_num(v);
    
    # Call to helper
    let result = computeValue(12, 4);
    print_num(result);
}
```
