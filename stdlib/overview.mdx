The Neutronium standard library provides essential built-in functions for I/O operations and mathematical calculations. These functions are available to all Neutronium programs through external declarations.

## Using stdlib functions

To use standard library functions in your Neutronium program, declare them with the `extern fn` keyword:

```neutronium
# I/O functions
extern fn print_c(char: int);
extern fn print_num(num: int);

# Math functions
extern fn mod(a: int, b: int) -> int;
extern fn abs(val: int) -> int;
extern fn min(a: int, b: int) -> int;
extern fn max(a: int, b: int) -> int;
extern fn pow(a: int, b: int) -> int;
```

Once declared, you can call these functions like any other function in your code.

## Available functions

### I/O functions

I/O functions allow you to print output to stdout:

- **print_c(char: int)** - Prints a single character to stdout
- **print_num(num: int)** - Prints an integer number to stdout

See [I/O functions](/stdlib/io) for detailed documentation.

### Math functions

Math functions provide common mathematical operations:

- **mod(a: int, b: int) -> int** - Returns a modulo b
- **abs(val: int) -> int** - Returns the absolute value
- **min(a: int, b: int) -> int** - Returns the minimum of two values
- **max(a: int, b: int) -> int** - Returns the maximum of two values
- **pow(a: int, b: int) -> int** - Returns a raised to the power of b

See [Math functions](/stdlib/math) for detailed documentation.

## Example usage

```neutronium
extern fn print_num(num: int);
extern fn pow(a: int, b: int) -> int;
extern fn mod(a: int, b: int) -> int;
extern fn max(a: int, b: int) -> int;

fn main(): {
    # Calculate 2^8
    let p = pow(2, 8);
    
    # Get remainder of p / 7
    let m = mod(p, 7);
    
    # Get the larger value
    let v = max(p, m);
    
    # Print the result
    print_num(v);
}
```
